How to Tie a Perfection Loop

Intermediate
🎣 Fishing🐟 Leader🎏 Fly Fishing

The perfection loop creates the strongest possible fixed end loop in monofilament or fluorocarbon — used for loop-to-loop leader connections and lure attachment where full line freedom is needed.

Written Instructions — Perfection Loop

  1. Start with line end Leave about 6 inches of tag end to work with.
  2. Form two overlapping loops Create a loop, then form a second smaller loop in front of (overlapping) the first.
  3. Pass tag end between the two loops Thread the tag end through the gap between both loops from back to front.
  4. Pull loop end to tighten Hold tag end, pull main line. Loop slides up and locks against the tag.
  5. Trim tag end Trim close to the knot. Attach via loop-to-loop or snap.

Tips for Tying a Perfection Loop

  • The two loops must be distinct — one clearly in front of the other — before passing the tag end through.
  • Stronger than a surgeon's loop because there are no sharp bends in the line.
  • Size the loop before tightening — once locked it cannot be adjusted.
  • This loop allows full 360° swivel movement — keeps lures swimming naturally.
